# CDSS Backend (FastAPI)\n\n## Endpoints\n- `GET /drugs/search` — Query params: `q` (required, min 2 chars), optional `limit`, `offset`. Returns paginated drugs matching name or generic name.\n- `GET /drugs/{id}` — Returns a single drug with therapeutic class, mechanism, side effects, diseases, and contraindications.\n- `GET /clinical/drugs-by-side-effect` — Query param `name`. Lists drugs causing the given side effect.\n- `GET /clinical/drugs-by-disease` — Query param `name`. Lists drugs indicated for the disease.\n- `GET /clinical/drug-contraindications` — Query param `condition`. Lists drugs contraindicated in the condition (e.g., Pregnancy).\n- `GET /clinical/drug-interactions` — Query param `drug`. Lists drugs that interact with the given drug (both directions).\n- `GET /clinical/drugs-by-ecg-effect` — Query param `effect`. Lists drugs associated with the ECG effect (e.g., QT prolongation).\n\n## Running locally\n1. Set database URL: `export CDSS_DATABASE_URL=postgresql+psycopg2://cdss_user:strongpassword@localhost/cdss_db` (or your DSN).\n2. Apply migrations: `alembic upgrade head`.\n3. Start API: `uvicorn app.main:app --reload`.\n\n## Notes\n- Models use PostgreSQL UUID primary keys and cascade deletes on relationship tables for safe deduplication.\n- Importer: run `python scripts/import_drugs.py` after placing the CSV (default `all_medicine databased.csv`).\n*** End Patch
